# edumaps-dojo

Dojo de mantenimiento y evolución de [Edumaps](https://edumaps.ar) — atlas histórico mundial interactivo.

## Modo
Orquestador — el código vive en los repos de Andrés. Este dojo gestiona sesiones, decisiones técnicas y memoria sobre esa plataforma.

## Estructura
```
docs/       documentación técnica y decisiones
scripts/    utilidades del dojo (sync-sensei, etc.)
memory/     contexto del proyecto (versionado)
.claude/    SENSEI, skills y configuración
```

## Cómo empezar

Abrir Claude Code en este directorio e invocar `/session-start`.
